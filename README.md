# Storm Control

## Usage

A collection of tools network tools I made for myself. 

You can access the site here: https://stormcontrol.net

Includes Tools for 

* IPv4 Subnetting
* Decimal/Hex/Binary converter
* Bits/Bytes/Megabytes etc converter
* MAC address lookup
* DNS lookup
* Whois lookup
